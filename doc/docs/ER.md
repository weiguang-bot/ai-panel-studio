  ┌─────────┬──────────┬────────────┬─────────────────────────────────────────────────────┐

  ├─────────┼──────────┼────────────┼─────────────────────────────────────────────────────┤
  │ Session │ 1 ──── N │ Consensus  │ 讨论结束后汇集 0~N 条共识要点                       │
  ├─────────┼──────────┼────────────┼─────────────────────────────────────────────────────┤
  │ Session │ 1 ──── N │ Divergence │ 讨论结束后汇集 0~N 条分歧要点                       │
  ├─────────┼──────────┼────────────┼─────────────────────────────────────────────────────┤
  │ Expert  │ 1 ──── N │ Transcript │ 每位专家发言 0~N 次；Host 发言时 speaker_id 为空    │
  └─────────┴──────────┴────────────┴─────────────────────────────────────────────────────┘
